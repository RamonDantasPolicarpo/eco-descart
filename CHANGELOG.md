# Changelog

Todas as mudanças notáveis neste projeto serão documentadas neste arquivo.

## [1.0.0] - 2026-04-09
### Adicionado
- Integração com a API do Google Gemini (Spring AI) para identificação de resíduos.
- Estrutura de CLI para interação com o usuário via terminal.
- Testes unitários com Mockito para simulação da inteligência artificial.
- Configuração de Linting com Checkstyle usando o padrão Google.
- Pipeline de CI configurada com GitHub Actions.

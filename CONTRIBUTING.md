# Como Contribuir

Obrigado por se interessar em contribuir com o EcoDescart!

## Passos para contribuir:
1. Faça um Fork do projeto
2. Crie uma Branch para sua Feature (`git checkout -b feature/NovaFuncionalidade`)
3. Faça o Commit de suas mudanças (`git commit -m 'feat: Adiciona nova funcionalidade'`)
4. Faça o Push para a Branch (`git push origin feature/NovaFuncionalidade`)
5. Abra um Pull Request

Certifique-se de que o código passa no Checkstyle (`./mvnw verify`) antes de abrir o PR.
